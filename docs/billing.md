# NexusMind 订阅与计量规则

## 1. 商业模式

- 免费版：用户自备 API Key，核心功能可用
- 订阅版：NexusMind 托管 AI 调用，默认每月 500 次
- 超额：按次计费或购买增量包

## 2. 计量单位

- 每次问答完成记一次调用
- 流式响应按一次完整会话计费

## 3. 额度控制

- 月度额度：套餐配额（如 500 次）
- 日度阈值：防滥用阈值（用于风控）
- 达到上限时返回明确提示并提供购买入口

## 4. Phase 5 实现状态

- 已落地订阅状态机：
  - 计划：`free | subscription`
  - 状态：`inactive | active | canceled | refunded`
- 已落地订阅校验与权限门控：
  - `NEXUSMIND_SUBSCRIPTION_VERIFY` 校验成功后激活订阅轨道
  - 问答入口统一执行门控，拦截未激活订阅、风控封禁、额度不足场景
- 已落地调用计数：
  - 完整问答会话成功后扣减 1 次
  - 订阅优先扣月度 500 次；月度耗尽后扣增量包
- 已落地超额链路：
  - 提供固定购买入口 URL
  - 支持增量包购买消息 `NEXUSMIND_BILLING_BUY_TOPUP`
- 已落地取消/退款：
  - `NEXUSMIND_BILLING_CANCEL` 标记“周期末取消”
  - `NEXUSMIND_BILLING_REFUND` 回收订阅权益并回到免费轨道

## 5. 风控与审计

- 高频保护：一分钟内调用次数异常触发风控拦截
- 审计日志：记录动作、时间、结果与最小必要元数据
- 安全要求：禁止记录 API Key、页面正文、模型响应原文

## 6. 入口与返回口径

- 查询计费状态：`NEXUSMIND_BILLING_STATUS`
- 统一返回口径：计划、状态、月度额度、增量包余额、风险状态、最近审计日志
- 达到上限时返回明确可执行提示并附购买入口
