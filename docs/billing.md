# NexusMind 订阅与计量规则

## 1. 商业模式

- 免费版：用户自备 API Key，核心功能可用
- 订阅版：NexusMind 托管 AI 调用，默认每月 500 次
- 超额：按次计费或购买增量包

## 2. 计量单位

- 每次问答完成记一次调用
- 流式响应按一次完整会话计费

## 3. 额度控制

- 月度额度：套餐配额（如 500 次）
- 日度阈值：防滥用阈值（用于风控）
- 达到上限时返回明确提示并提供购买入口

## 4. Phase 1 实现状态

- 已提供配额判断基础接口 `canInvoke`
- 完整订阅鉴权、计数、风控在 Phase 5 实施
