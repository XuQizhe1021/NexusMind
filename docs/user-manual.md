# NexusMind 用户使用手册（Phase 7）

## 1. 安装与加载

1. 执行 `npm install`
2. 执行 `npm run build`
3. 打开 Chrome 或 Edge 扩展管理页并启用开发者模式
4. 选择 `apps/extension/dist` 目录加载扩展

## 2. 核心功能路径

### 2.1 AI 伴侣（当前页与跨页问答）

1. 打开侧边栏，在“当前页 AI 问答”输入问题
2. 点击“跨页图谱问答”
3. 观察流式答案输出，必要时点击“中断回答”
4. 点击答案中的 `[S1]/[S2]` 或来源卡片执行正文定位

### 2.2 知识织网（图谱）

1. 在“知识织网（Phase 2）”点击“收录当前页”
2. 点击“刷新图谱统计”确认实体/关系/页面数量
3. 输入关键词并“搜索并回溯”查看跨页关系

### 2.3 页面重写

1. 在“页面重写（Phase 3）”选择意图（学习/摘要/去干扰）
2. 点击“应用页面重写”
3. 点击“一键还原”恢复页面
4. SPA 站点路由切换后会自动回滚

### 2.4 商业化链路

1. 在“商业化（Phase 5）”点击“刷新订阅状态”
2. 输入服务端签名 JWT 并点击“校验并激活订阅”
3. 额度不足时通过“购买增量包”补充
4. 需要停用时使用“申请取消订阅”或“申请退款”
5. 风控场景下使用“提交人工复核”

## 3. 常见问题

- 订阅校验失败：请确认 Token 为服务端签发的 RS256 JWT，且未过期。
- 页面重写无变化：确认当前页面正文可识别，或切换重写意图后重试。
- 图谱无结果：先执行“收录当前页”，再使用页面中实际出现的关键词检索。
- 流式中断：网络波动时会自动重连；持续失败可重试问题或刷新页面。

## 4. 隐私与安全提示

- API Key 仅本地加密存储，不以明文写入磁盘。
- 未经用户触发不会主动调用 AI 服务。
- 审计日志不记录 API Key、页面正文、模型响应原文。
