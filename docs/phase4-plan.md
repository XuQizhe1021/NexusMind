# Phase 4 实施计划（高级问答）

## 1. 实施计划

1. 设计跨页图谱问答编排：以问题关键词检索图谱证据，构建证据上下文后再触发答案生成，并保留来源回链。
2. 升级侧边栏问答交互：支持答案分块渲染、来源列表展示、引用标记点击定位正文。
3. 升级流式输出与中断控制：引入长连接消息通道，支持增量输出、取消请求与并发隔离。
4. 补齐 Phase 4 集成/E2E 测试：覆盖图谱证据检索、流式解析与中断、答案引用解析与定位协议。

## 2. 里程碑

- M1：图谱服务输出问答证据集（实体、关系、页面摘要与来源链接）。
- M2：后台完成“检索证据 -> 生成答案 -> 回链来源”的问答编排并可流式回传。
- M3：侧边栏完成答案引用渲染与正文高亮联动，支持用户主动中断。
- M4：测试、构建、类型检查全部通过，文档与回滚方案更新完成。

## 3. 风险清单

- 证据命中风险：问题关键词与实体归一键不一致可能导致召回不足。
- 引用一致性风险：模型输出引用标记可能与证据编号偏差，需前端容错渲染。
- 流式稳定性风险：浏览器后台长连接在异常断开时可能导致 UI 状态悬挂。
- 正文定位风险：复杂 DOM 或动态站点下，片段精确定位可能退化为近似匹配。

## 4. 验收口径（DoD）

- 功能完成：实现跨页图谱问答编排、答案回链来源、答案片段正文定位、流式输出与中断控制。
- 测试通过：新增单元/集成/E2E 测试并全量通过。
- 性能达标：页面重写既有 1 秒目标不回退；图谱 2000 节点检索仍流畅；问答支持增量流式响应。
- 文档更新：README、architecture、assumptions、changelog 与 Phase 4 计划同步更新。
- 可回滚明确：问答请求可中断，正文高亮可清理，异常断流后 UI 可恢复空闲状态。
