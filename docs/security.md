# NexusMind 安全与隐私策略

## 1. 数据存储原则

- 默认本地存储：配置与运行状态使用 `chrome.storage.local`
- 不上传浏览记录与页面内容到 NexusMind 自建服务
- 仅在用户主动点击提问时调用第三方 AI 服务

## 2. API Key 保护

- API Key 不以明文写入存储
- 保存前使用 AES-GCM 加密，存储密文、IV、Salt
- 解密仅发生在后台服务中，用于实时请求 AI

## 3. 敏感信息日志策略

- 禁止输出 API Key、原文页面数据、Token 等敏感信息
- 错误日志仅返回最小必要信息（状态码、错误类型）

## 4. 供应商边界

- 当前仅向用户指定 AI Provider 发起请求
- 不将用户内容转发到未声明服务
- 后续接入多 Provider 时保持同一审计策略
