# NexusMind

NexusMind 是一个本地优先、面向知识工作流的智能浏览器扩展项目，目标是把浏览器从信息容器升级为思维伙伴。

## 当前状态

- 版本：`v0.4.0`（Phase 6 稳定性能力已完成开发与验证）
- 已完成：
  - MV3 扩展骨架（background/content script/side panel）
  - 当前页面 AI 问答闭环（用户提问 → 提取页面文本 → 调用模型 → 返回答案）
  - 基础设置页（Provider / Model / API Key / 隐私模式 / 成本阈值）
  - 图谱基础能力：IndexedDB（Dexie）实体/关系/Page 模型
  - 跨页实体归一与关系写入流水线（支持重复收录回滚）
  - 侧边栏图谱最小可视化 + 搜索回溯
  - 图谱单元/集成/性能测试（2000 节点）
  - 页面重写能力：学习/摘要/去干扰三意图
  - 站点默认重写意图配置（按域名覆盖全局默认意图）
  - DOM 可逆重排与一键还原（含 SPA 路由切换自动回滚）
  - 页面重写测试：规则测试、回滚稳定性测试、1 秒性能基准
  - 跨页图谱问答编排（检索证据 → 生成答案 → 来源回链）
  - 答案引用联动定位（[S1]/[S2] 引用点击与正文高亮）
  - AI 流式输出与中断控制（增量返回、用户可取消）
  - Phase 4 测试：图谱证据集成测试、流式与中断集成测试、正文高亮 E2E 测试
  - 订阅校验与权限门控（免费/订阅双轨）
  - 订阅月度 500 次额度控制与调用计量
  - 超额计费入口与增量包购买链路
  - 退款/取消流程与风控审计日志（不记录敏感明文）
  - 流式断流监控与自动重连（最多 2 次）
  - 稳定性看板（流式成功率/重连恢复/最近错误摘要）
  - 计费 requestId 幂等去重（避免重连重扣）
  - 风控分层（none/degraded/blocked）+ 白名单 + 人工复核入口
  - Phase 6 固定回归集与发布前稳定性门禁

## Monorepo 目录

```text
apps/
  extension/   # 浏览器扩展 (MV3)
  web/         # 官网/账户体系（预留）
packages/
  core/        # 领域模型、Schema、加密与设置
  ai/          # AI Provider 适配
  graph/       # 图谱引擎与检索（Phase 2 已落地）
  billing/     # 订阅、额度、风控与退款流程
  ui/          # 复用 UI（占位）
docs/          # 架构、安全、计费、发布、假设、变更
tests/         # 单元/集成测试
```

## 本地开发

1. 安装依赖

```bash
npm install
```

2. 类型检查

```bash
npm run typecheck
```

3. 运行测试

```bash
npm test
```

4. 构建扩展

```bash
npm run build
```

5. 在 Chrome/Edge 加载扩展
   - 打开扩展管理页面
   - 启用开发者模式
   - 选择 `apps/extension/dist` 目录进行加载

## Phase 2 使用方式（知识织网）

1. 打开任意页面并唤起 NexusMind 侧边栏
2. 在“知识织网（Phase 2）”区域点击“收录当前页”
3. 点击“刷新图谱统计”查看实体/关系/页面计数
4. 在“图谱搜索”中输入关键词并点击“搜索并回溯”
5. 查看可视化节点与关系路径回溯文本结果

## Phase 3 使用方式（页面重写）

1. 打开任意页面并唤起 NexusMind 侧边栏
2. 在“页面重写（Phase 3）”区域选择意图（学习/摘要/去干扰）
3. 点击“应用页面重写”执行 DOM 重排
4. 点击“一键还原”可恢复原始页面结构
5. 在“基础设置”区域可设置全局默认重写意图
6. 点击“设为当前站点默认意图”并保存设置，可按域名覆盖默认意图

## Phase 4 使用方式（高级问答）

1. 在“当前页 AI 问答”输入问题，点击“跨页图谱问答”
2. 侧边栏会先检索图谱证据，再进行流式答案生成
3. 答案中的 `[S1] [S2]` 可点击定位当前页面正文片段
4. 来源卡片支持“定位正文”与“打开来源”操作
5. 生成过程中点击“中断回答”可立即取消请求

## Phase 5 使用方式（商业化）

1. 打开侧边栏“商业化（Phase 5）”，点击“刷新订阅状态”查看当前账期额度
2. 输入 `nm_sub_xxx` 形式 Token 并点击“校验并激活订阅”
3. 问答调用会自动执行订阅权限门控与计量（按完整会话计 1 次）
4. 月度 500 次额度耗尽后，通过“购买增量包”补充调用次数
5. 需要停用时点击“申请取消订阅”，退款场景填写退款单号并点击“申请退款”
6. 审计日志仅记录动作与状态，不记录 API Key 与页面正文

## Phase 6 使用方式（稳定性）

1. 在“当前页 AI 问答”发起跨页问答，网络抖动时会显示“正在重连”提示
2. 在“商业化（Phase 5）”可提交人工复核，生成风控复核工单号
3. 订阅校验支持白名单 Token（`nm_sub_vip_` 前缀）以绕过高频风控拦截
4. 在“稳定性看板（Phase 6）”点击“刷新稳定性看板”查看会话成功率与最近错误
5. 发布前按 `docs/phase6-regression-report.md` 执行固定回归集并记录结果

## 已实现关键约束

- 本地优先：设置与业务状态保存在 `chrome.storage.local`
- API Key 加密：保存前进行 AES-GCM 加密（详见 `docs/security.md`）
- 用户触发原则：仅在用户主动点击“提问”时请求 AI
- 图谱收录触发原则：仅在用户主动点击“收录当前页”时写入图谱
- 页面重写触发原则：仅在用户主动点击“应用页面重写”时执行 DOM 改写
- 图谱问答触发原则：仅在用户主动点击“跨页图谱问答”时请求 AI
- 计费触发原则：仅在用户主动触发问答时执行计量与扣减
- 可扩展边界：UI 与 AI Provider 通过消息与接口解耦

## 常见问题

- `请先在设置中保存 API Key`
  - 先在侧边栏设置页填写并保存 API Key，再执行问答
- `当前页面文本提取失败`
  - 当前标签页可能尚未注入 content script，刷新页面后重试
- `图谱搜索无结果`
  - 先执行“收录当前页”，并确认关键词与页面中的实体文本一致
- `页面重写后样式残留`
  - 点击“一键还原”；若站点为 SPA 路由场景，切换路由会自动执行回滚
- `引用定位未命中`
  - 页面结构可能已动态变化，建议先刷新页面后重试“定位正文”
- `订阅校验失败：Token 无效`
  - 当前仅接受 `nm_sub_` 前缀的订阅校验 Token
- `本月 500 次订阅额度已用尽`
  - 在“商业化（Phase 5）”购买增量包后重试
- `连接抖动，正在重连`
  - 表示后台正在自动重试，若连续失败会返回最终错误摘要
