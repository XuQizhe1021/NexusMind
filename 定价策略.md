# NexusMind 收费方案

## 1. 收费模式概述

NexusMind 采用 **“基础功能免费 + 高级功能订阅”** 的模式，同时支持 **用户自备 API Key** 的完全免费路径。  
- **自备 API Key**：用户绑定自己的 OpenAI/Claude/Gemini 等 API Key，可免费使用所有插件功能（包括知识图谱、页面重写、智能问答），仅需自行承担 API 调用费用。  
- **订阅服务**：用户无需自备 Key，由 NexusMind 提供封装好的 AI 服务（按月度/年度订阅），适合希望省去 API 管理成本的用户。  

**薄利多销策略**：定价远低于市场同类 AI 工具，以低价吸引大量用户，通过规模化分摊 AI 调用成本，同时鼓励用户升级到年度订阅以提高留存。

## 2. 免费版功能 vs 付费版功能

### 2.1 免费版（自备 API Key）
- ✅ 所有核心功能：知识图谱构建、意图重写、AI 问答（基于用户自己的 API Key）  
- ✅ 本地知识图谱存储（IndexedDB）  
- ✅ 手动添加/编辑实体与关系  
- ❌ 无 AI 调用额度（用户自行承担费用）  
- ❌ 不支持多模态图片理解（需用户 Key 支持）  
- ❌ 无主动提醒与智能推荐  

### 2.2 付费订阅版（使用 NexusMind 云服务）
**包含以下所有功能，并额外提供：**
- ✅ 每月 500 次 AI 调用（覆盖普通总结、实体提取、问答等）  
- ✅ 多模态图片理解（自动识别图表、截图）  
- ✅ 主动提醒（基于知识图谱的智能推荐）  
- ✅ 优先技术支持与功能更新  
- ✅ 无限知识图谱节点存储（免费版节点数限制为 200，付费版无限制）  
- ✅ 团队协作功能（未来规划：共享图谱、协作编辑）  

**超出调用次数**：  
- 额外调用按量计费：$0.01/次（或购买增量包：$4.99/500 次）

## 4. 收费技术实现

### 4.1 支付与订阅管理
- **支付网关**：采用 **Paddle** 或 **Stripe**，支持全球信用卡、PayPal 等支付方式。  
  - Paddle 优势：自动处理全球税务（VAT/GST），适合跨境销售。  
  - Stripe 优势：API 成熟，可定制性强。  
- **订阅状态存储**：用户在插件内登录账号（邮箱 + 密码），后端记录订阅状态、调用次数。  
- **许可证验证**：插件启动时通过 HTTPS 请求验证用户订阅状态（可缓存 24 小时，避免频繁网络请求）。  
- **调用计数**：每次 AI 调用（非流式完成）计数一次，存储在后端数据库，实时扣减。

### 4.2 用户鉴权流程
1. 用户在插件内注册/登录（邮箱 + 密码）。  
2. 登录成功后，获取 JWT token 存储于 `chrome.storage.local`。  
3. 每次调用插件云服务时，携带 token 进行鉴权与用量检查。  
4. 若超出月度额度，返回错误提示，引导用户购买增量包或升级套餐。

### 4.3 防滥用机制
- **速率限制**：同一用户每分钟不超过 10 次调用（防止恶意刷量）。  
- **异常检测**：若单日调用量异常（如超过 500 次），触发风控，暂时冻结并通知用户。  
- **公平使用原则**：所有付费用户共享云服务资源，但调用上限按套餐严格限制。

### 4.4 退款与取消政策
- **14 天内无条件退款**：通过支付渠道自动处理。  
- **取消订阅**：用户在支付平台取消后，订阅状态在下一个计费周期失效，已付费月份仍可继续使用。

## 5. 薄利多销的可行性分析

| 因素                  | 分析                                                         |
| --------------------- | ------------------------------------------------------------ |
| **目标用户基数**      | 全球知识工作者、学生、开发者数量庞大，插件市场同类产品较少，市场空间大。 |
| **定价对比**          | 同类 AI 工具（如 Monica、Merlin）月费约 $8–$15，NexusMind $2.99 具有显著价格优势。 |
| **AI 成本**           | 若用户平均每月使用 300 次调用，单次成本约 $0.003（按 Claude 3 Haiku 估算），总成本约 $0.9/用户/月，毛利率约 70%。 |
| **用户留存**          | 低价可降低用户决策门槛，提高付费转化率；年度订阅可锁定用户，降低流失。 |
| **自备 API Key 路径** | 为不愿意付费的用户提供完全免费路径，但限制部分高级体验，促使部分用户转为订阅。 |
| **规模化效应**        | 用户量越大，AI 调用可批量采购（如使用企业级 API 折扣），进一步降低成本。 |
